```
Usage: SET AUTO_WHOWAS [ON|OFF|TOGGLE]
  If set to ON, this will automatically generate a 
    WHOWAS <nickname>
  for any nick command that results in 
    *** <nickname>: No such nickname
  If OFF, you will not be bothered with WHOWAS. 
```
