```
Usage: SET BIND_LOCAL_DCCHOST [ON|OFF]
  If ON, then all dcc requests you send will be bound to
  a local interface according to the DCCHOST you have set
  (-H switch on command line).

See also:
  SET DCCHOST
  DCC CHAT
  DCC SEND
```
